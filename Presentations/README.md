# Presentations

This folder contains two presentations delivered during the course:

1. **CSV File Presentation** – Covers the basics of CSV files, their structure, usage in data storage and analysis, and practical examples in Python for reading, writing, and manipulating CSV data.  

2. **Upwork Presentation** – Provides an overview of the Upwork platform, including freelancing opportunities, profile creation, job application strategies, and tips for succeeding as a freelancer.  

These presentations showcase practical knowledge and communication skills developed during the course.
