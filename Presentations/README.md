# Presentations

This folder contains two presentations delivered during the course:

1. **CSV File Presentation** – A presentation on CSV files, covering their structure, usage in data storage and analysis, and practical Python examples for reading, writing, and manipulating CSV data.  

2. **Upwork Presentation** – A presentation on the Upwork platform, including freelancing opportunities, profile creation, job application strategies, and tips for succeeding as a freelancer.  

These presentations demonstrate the knowledge shared and communication skills applied during the course.
