# Python Basics Lab

This repository contains **beginner-level Python lab exercises** for the course **Introduction to AI Programming**. The lab focuses on fundamental Python concepts, including:

- Using the `print()` function to display output  
- Understanding basic data types: integer, float, string, boolean  
- Creating and using variables  
- Taking user input  
- Checking variable types with `type()`  

These exercises are designed for beginners to get hands-on practice with Python syntax and basic programming logic. The code snippets are simple, easy to understand, and ideal for building a strong foundation in Python programming.
