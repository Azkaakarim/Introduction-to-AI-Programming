## Lab 2: Python Basic Data Types and File Handling  

This folder contains the lab work for **Lab 2**, focusing on fundamental Python programming concepts.

### Topics Covered
- Basic Python data types  
- Input/output operations and formatting  
- File handling using the CSV module  
- Built-in functions for counting and condition checking  
- Directory navigation and file filtering  

### Objective
To develop practical skills in working with Python data types, formatting outputs, handling CSV files, applying built-in functions, and navigating directories, providing a foundation for further AI programming tasks.
