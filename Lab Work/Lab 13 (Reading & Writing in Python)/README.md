# Lab 13 – Reading and Writing in Python

This repository contains Lab 13 tasks for practicing **file handling in Python**.  
The lab focuses on reading, writing, and organizing data in **CSV and text files** for electrical engineering applications.

## Objective

Learn to handle files in Python to:

- Record hourly load demand
- Log solar energy generation
- Maintain transformer maintenance reports
- Store student energy audit data
- Manage electricity billing records
- Record substation fault events
- Process and organize CSV data

## Contents

- **load_demand.csv** – Hourly load demand data  
- **solar_generation.csv** – Daily solar energy generation records  
- **transformer_maintenance.txt** – Maintenance activity log  
- **student_audit.csv** – Student energy audit records  
- **billing_data.csv** – Monthly electricity billing data  
- **fault_log.csv** – Substation fault events  
- **organized_energy_data.csv** – Processed CSV records
