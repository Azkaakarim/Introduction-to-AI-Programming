## Lab 7: Dictionary  

This folder contains the lab work for **Lab 7**, focusing on Python dictionaries and their practical applications.

### Topics Covered
- Creating, accessing, and updating dictionary entries  
- Nested dictionaries and complex data structures  
- Deleting entries and using `.pop()`  
- Checking key existence with `in`  
- Using built-in dictionary methods: `.keys()`, `.values()`, `.items()`, `.get()`, `.update()`  

### Objective
To develop practical skills in organizing, retrieving, and manipulating structured data using Python dictionaries, applied to scenarios like smart home energy meters, student records, weather monitoring, inventory systems, and patient monitoring in engineering and AI contexts.
