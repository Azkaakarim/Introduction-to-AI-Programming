This folder contains **Assignment 1** of the course **Introduction to AI Programming**. The assignment focuses on working with CSV data, performing calculations, and presenting results using Python.

## Assignment Tasks

1. Load the provided CSV file manually using the `csv` module (without using pandas).  
2. Store the **consumption** column in a list of floats and compute:  
   - Total consumption  
   - Average consumption  
   - Maximum and minimum values  
3. Use a **list comprehension** to convert hourly consumption into kWh.  
4. Print all results in a clean and organized format.

## Purpose

The assignment is designed to help students practice handling CSV files, applying list operations, performing basic data analysis, and presenting results in a structured manner using Python.
