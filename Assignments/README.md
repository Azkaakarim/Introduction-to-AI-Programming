# Assignments – Introduction to AI Programming

This folder contains all assignments for the course **Introduction to AI Programming**. It serves as a centralized location for organizing and maintaining all assignment-related work.

## Folder Structure

The `Assignments` folder is organized into subfolders for each assignment:

- `Assignment 1` – Contains all notebooks and documentation related to Assignment 1.
- `Assignment 2` – Contains all notebooks and documentation related to Assignment 2.
- `Assignment 3` – Contains all notebooks and documentation related to Assignment 3.

Each assignment folder may include:

- Jupyter Notebook files (`.ipynb`)
- Supporting documents or data files
- Markdown or text reports explaining tasks, objectives, and results

## Purpose

This folder aims to:

- Maintain a clear and organized repository of all assignment work
- Facilitate easy navigation for students, instructors, and reviewers
- Provide a structured record of practical exercises completed as part of the course
